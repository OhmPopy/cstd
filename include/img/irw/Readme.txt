

此目录放置图像读写模块的所有代码，它们的用途如下：
____________________________________________________________________________

	CVS	- 本目录的CVS文件目录
	ani	- ANI图象格式读写插件
	bmp	- BMP图象格式读写插件
	cur	- CUR图象格式读写插件
	emf	- EMF图象格式读写插件
	gif	- GIF图象格式读写插件
	ico	- ICON图象格式读写插件
	irwcpp	- ISee图象读写引擎 C++ 语言接口模块
	iseeio	- ISeeIO系统
	iseeirw	- ISee图象读写引擎及 C 语言接口模块
	iseemem	- ISee内存防护系统
	iseetsvc- ISee图象读写引擎测试软件
	jbig	- JBIG图象格式读写插件
	jp2	- JPEG2000图象格式读写插件（未完成）
	jpeg	- JPEG图象格式读写插件
	lcd	- LCD图象格式读写插件
	pcd	- PCD图象格式读写插件
	pct	- PCT图象格式读写插件
	pcx	- PCX图象格式读写插件
	pics	- PICS图象格式读写插件
	pix	- PIX图象格式读写插件
	png	- PNG图象格式读写插件
	pnm	- PNM(PBM,PGM,PPM)图象格式读写插件
	psd	- PSD图象格式读写插件
	ras	- RAS图象格式读写插件
	rla	- RLA图象格式读写插件
	rle	- RLE图象格式读写插件
	sgi	- SGI图象格式读写插件
	tga	- TGA图象格式读写插件
	tiff	- TIFF图象格式读写插件（存在BUG）
	wmf	- WMF图象格式读写插件
	xbm	- XBM图象格式读写插件
	xpm	- XPM图象格式读写插件
